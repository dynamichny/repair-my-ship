# Repair My ‘Ship #
Shortcut for “Repair My Relationship”. It is a forum where users can post about their relationship issues, and others can chime in with advice on how to fix their problems.

Idea from freeCodeCamp.org.

### Used technology: 
* VueJS
* VueX
* vue-router
* Firebase
